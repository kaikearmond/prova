# Dashboard SUS - Streamlit

Dashboard criado para reproduzir as telas enviadas:

1. Quantidade e Valor Total
2. Total de Procedimentos e Gastos
3. Mapa
4. Procedimentos Interativos

## Como rodar

No terminal, entre na pasta do projeto e execute:

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Arquivos

- `app.py`: aplicação principal em Streamlit.
- `data/CIA014_SUS_Prova_2.xlsx`: base de dados usada no dashboard.
- `assets/iesb_logo.png`: logo usada no cabeçalho.
- `assets/referencias/`: prints enviados como referência visual.
